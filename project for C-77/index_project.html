Why VS Code is good and how to set it up.

Visual Studio Code (VS Code) is a very user friendly IDE (Integrated Developement Environment). It is available on Mac OS, Linux and Windows. Many Web Developers and Game Developers use it. VS Code has a lot of in-built features and commands which makes it really easy to use. One can set it up just bu browsing it on any Search Engine and downloading it from there.



What is the console screen?

The console screen is a window that shows us what all the computer is processing while the code runs. It helps the person to spot mistakes and confirm the code. One can access it by right clicking on the screen and selecting the inspect option and then choosing the console option.
